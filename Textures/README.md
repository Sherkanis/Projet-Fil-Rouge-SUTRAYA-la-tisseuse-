Ici on mettra les dossiers textures des assets. 



Il y aura un dossier texture portant le nom de l'asset, puis à l'intérieur des dossiers portant le nom du matériaux présent dans la scène selon la nomenclature suivante : 



M\_(nom du matériaux)



A l'intérieur de chaque dossier de matériaux, les maps de textures avec la nomenclature suivante : 



T\_(nom de l'asset)\_(nom du matériaux)\_(nom de la map : ex: basecolor, normal…)
